    üìò Hugo Configuration Guide

    Hugo uses a single configuration file, typically named hugo.yaml, located in the root of your site. This file lets you define how your site behaves‚Äîfrom menus and logos to sidebars and footers.

    üß≠ Navigation Menu

    The top-right navigation menu is defined in the menu.main section of your hugo.yaml.

    Example:

    menu:
  main:
    - name: Documentation
      pageRef: /docs
      weight: 1
    - name: Blog
      pageRef: /blog
      weight: 2
    - name: About
      pageRef: /about
      weight: 3
    - name: Search
      weight: 4
      params:
        type: search
    - name: GitHub
      weight: 5
      url: "https://github.com/imfing/hextra"
      params:
        icon: github

    Supported Menu Item Types:

    • Internal Link (using pageRef)
    - name: Docs
  pageRef: /docs
    • External Link (using url)
    - name: GitHub
  url: "https://github.com"
    • Search Bar
    - name: Search
  params:
    type: search
    • Icon Support
    - name: GitHub
  params:
    icon: github

    Use weight to control menu order.

    üñºÔ∏è Logo & Site Title

    Customize the logo and site title in your hugo.yaml under params.navbar.

    • You can also define dark mode and link behavior.
